# Ch-ISS
This is a simple Chess Game written in python

## Requirements:
- Linux Shell

- Python3:
  - Run this in terminal:
    
    `sudo apt-get install python3`
  
## Usage:
- Clone this repo: 
  - `git clone https://github.com/MohsinMamoon/Ch-ISS.git`
  
- `cd` into the directory:
  - `cd Ch-ISS`
  
- Run `run.py`:
  - `python3 run.py`
